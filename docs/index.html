🤖 Agent Orchestration Layer
  • LangGraph StateGraph with MemorySaver checkpointer
  • Supervisor pattern — LLM routes to 1–3 agents
  • Parallel Send fanout for multi-agent queries
  • Annotated[list, add] reducer on agent_results
  • astream_events for real-time token streaming to WS
🔌 Adapter Layer
  • Abstract base classes: MarketDataAdapter, NewsAdapter
  • AdapterRegistry — priority-sorted, Redis-cached
  • Automatic fallthrough on adapter failure
  • YFinance (primary) → AlphaVantage (fallback)

LangGraph State Schema

All graph nodes read from and write to a single shared FinanceStateDict. The agent_results field uses an add reducer so parallel agents safely append without overwriting each other.

class FinanceStateDict(TypedDict, total=False):
    # Conversation history — LangChain add_messages reducer
    messages:          Annotated[list[BaseMessage], add_messages]
    user_id:           str
    conversation_id:   str

    # Routing — set by supervisor_route node
    intent:            str | None           # "user wants stock price"
    selected_agents:   list[str]             # ["market", "portfolio"]
    rag_categories:    list[str]             # ["market", "portfolio"]
    routing_reason:    str | None
    needs_followup:    bool                  # True if confidence < 0.6

    # Data — filled during execution
    retrieved_docs:    list[dict]            # RAG chunks from ChromaDB
    tool_outputs:      dict[str, Any]       # arbitrary tool call results
    agent_results:     Annotated[list[dict], add]  # parallel-safe

    # Output — filled by synthesizer or persist_turn
    final_response:    str | None
    final_citations:   list[dict]
    error:             str | None

WebSocket Streaming Protocol

The gateway streams typed JSON frames over a WebSocket connection at /api/chat/ws?conversation_id=<id>&token=<jwt>.

agent_start — emitted when a specialist agent begins processing
{"type": "agent_start", "agent": "portfolio"}
token — streaming token chunk from the LLM
{"type": "token", "delta": "Your portfolio consists of..."}
agent_end — emitted when an agent finishes
{"type": "agent_end", "agent": "portfolio"}
citation — RAG source reference
{"type": "citation", "source_title": "Diversification Guide", "source_url": "...", "snippet": "..."}
final — complete response with all metadata
{"type": "final", "message_id": "uuid", "content": "...", "agents_used": ["portfolio","market"], "citations": [...]}
error — if graph execution fails
{"type": "error", "message": "LLM timeout"}

Reconnect Strategy (client-side)

// Exponential backoff: 1s → 2s → 4s → ... → 30s max, 8 attempts
const BASE_MS = 1000;
const MAX_MS  = 30000;
const MAX_ATT = 8;

ws.onclose = (ev) => {
  if (ev.code !== 1000 && attempts < MAX_ATT) {
    const delay = Math.min(BASE_MS * 2 ** attempts, MAX_MS);
    attempts++;
    reconnectTimer = setTimeout(connect, delay);
  }
};
// Clean close on component unmount — code 1000 prevents reconnect
ws.close(1000, "component unmounted");

Registry Priority Fallthrough

async def get_quote(self, symbol):
    # 1. Check Redis cache first
    cached = await self._cache.get(key)
    if cached: return Quote.model_validate_json(cached)

    # 2. Try adapters in priority order
    for name in self._market_priority:
        try:
            quote = await self._market[name].get_quote(symbol)
            await self._cache.set(key, quote.model_dump_json(), ex=60)
            return quote
        except Exception:
            log_warning("adapter failed, trying next")

    raise NoAdapterAvailable(symbol)

Key Implementation Patterns

1. Multi-Provider LLM Factory

# src/core/llm.py — single function, zero agent code changes when switching providers
def get_llm(temperature=0.3, streaming=False):
    provider, model = settings.llm_provider, settings.llm_model
    if provider == "anthropic":
        from langchain_anthropic import ChatAnthropic
        return ChatAnthropic(model=model, streaming=streaming,
                             api_key=settings.anthropic_api_key.get_secret_value())
    elif provider == "openai":
        from langchain_openai import ChatOpenAI
        return ChatOpenAI(model=model, streaming=streaming, ...)
    elif provider == "google":
        from langchain_google_genai import ChatGoogleGenerativeAI
        return ChatGoogleGenerativeAI(model=model, ...)
    elif provider == "bedrock":
        from langchain_aws import ChatBedrock
        return ChatBedrock(model_id=model, region_name=settings.aws_region)
    raise ValueError(f"Unknown LLM_PROVIDER: {provider}")

2. Parallel Agent Fanout with Safe State Reduction

# graph.py — Send API dispatches agents in parallel
def _fanout_to_agents(state: dict) -> list[Send]:
    selected = state.get("selected_agents", ["qa"])
    return [Send(f"{agent}_agent", state) for agent in selected]

# state.py — Annotated add reducer: parallel appends never overwrite
class FinanceStateDict(TypedDict, total=False):
    agent_results: Annotated[list[dict], add]  # <-- thread-safe

# After all agents complete, route based on count
def _after_agents(state: dict) -> str:
    if len(state["selected_agents"]) > 1 and len(state["agent_results"]) >= len(state["selected_agents"]):
        return "synthesizer"   # merge multiple answers
    return "persist_turn"       # single agent: pass through

3. Dependency Injection into Graph Nodes

# graph.py — wrap each node to embed registry and chroma_store
def _inject(fn, registry, chroma_store):
    async def wrapper(state: dict) -> dict:
        enriched = dict(state)
        if registry:     enriched["__registry__"] = registry
        if chroma_store: enriched["__chroma_store__"] = chroma_store
        return await fn(enriched)
    return wrapper

# Usage: wrap every agent node with injected dependencies at graph build time
g.add_node("market_agent", _inject(market_agent, registry, chroma_store))

4. WebSocket astream_events Serializer

# ws.py — forward LangGraph events as typed WS frames
async for event in graph.astream_events(state, config=config, version="v2"):
    kind, name = event["event"], event["name"]

    if kind == "on_chain_start" and name.endswith("_agent"):
        await send(ws, {"type": "agent_start", "agent": name[:-6]})

    elif kind == "on_chat_model_stream":
        delta = event["data"]["chunk"].content
        if delta:
            final_response += delta
            await send(ws, {"type": "token", "delta": delta})

    elif kind == "on_chain_end" and name == "persist_turn":
        for c in event["data"]["output"]["final_citations"]:
            await send(ws, {"type": "citation", **c})

await send(ws, {"type": "final", "content": final_response, ...})

Testing

Test Strategy

⚡ Unit Tests

Pure logic with mocks. No network, no DB, no LLM API calls. Tests run in <1 second. Covers:

  • Security helpers (bcrypt, JWT)
  • In-memory cache (TTL buckets)
  • Adapter registry (priority, caching, fallthrough)
  • Supervisor routing (JSON parsing, fallback, filtering)
🔗 Integration Tests

Full HTTP request/response via FastAPI TestClient. Uses SQLite test DB + in-memory cache (no Docker required). Covers:

  • Auth flow: register, login, /me, 409/401/422 error cases
  • Portfolio CRUD: create, list, get, delete
  • Holdings CRUD: add, list, delete
  • Cross-user access control (404 for other user's resources)

Testing Notes

What's Mocked

  • LLM calls — @patch("src.agents.supervisor.get_llm") with AsyncMock; no API key required for unit tests
  • Market adapters — MagicMock with AsyncMock methods; no yfinance network calls
  • Database — overridden with SQLite via os.environ at conftest level; tables auto-created by lifespan
  • Cache — _InMemoryCache (no Redis container needed)

What's Not Mocked (Integration)

  • FastAPI routing, middleware, dependency injection
  • SQLAlchemy ORM queries on SQLite
  • bcrypt hashing and JWT encode/decode
  • Pydantic validation (422 responses)
  • Cross-user access control logic
